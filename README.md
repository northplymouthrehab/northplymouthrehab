# North Plymouth Rehabilitation Clinic website

Static website ready for GitHub and Netlify.

## Edit these first
- Replace placeholder emails, phone numbers and booking links.
- Add images inside `assets/images/` and update the placeholder sections in the HTML.
- Update prices in `prices.html`.
- Replace testimonial placeholder once you have permission.

## Deploy with Netlify
1. Create a GitHub repository.
2. Upload these files.
3. In Netlify, choose Add new site > Import from GitHub.
4. Select the repository.
5. Leave build command blank and publish directory as `/`.
