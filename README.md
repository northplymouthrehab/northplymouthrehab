# North Plymouth Performance & Rehabilitation website

Complete static website ready for GitHub + Netlify.

## Files included
- `index.html` — homepage with CrossFit Cann Woods hero image and Sarah profile card
- `about.html` — about/clinician page with Sarah photo and Vicki placeholder
- `services.html` — services page with Sarah image on golf rehabilitation card
- `prices.html` — pricing page
- `contact.html` — contact/enquiry page with Netlify-ready form markup
- `style.css` — all visual styling
- `script.js` — mobile menu and scroll reveal animations
- `assets/images/logo.png` — business logo
- `assets/images/crossfit-cann-woods.png` — homepage hero image
- `assets/images/sarah-martin-golf-rehab.png` — Sarah Martin profile image

## How to use
Upload the full contents of this folder to a new GitHub repository.

Then in Netlify: Add new site → Import from GitHub → select this repository → Deploy.

No build command is needed. Publish directory should be left blank or set to `/`.

## Editing later
Change text directly in the `.html` files. Replace images by keeping the same filenames in `assets/images/`.
